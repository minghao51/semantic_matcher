# Tech Stack

## Languages & Runtime

- **Python 3.9+** - Main development language (supports 3.9, 3.10, 3.11, 3.12)
- **UV** - Package management and virtual environment
- **Hatch** - Build system and packaging

## Core Frameworks & Libraries

### Deep Learning & ML
- **PyTorch** - Deep learning framework
- **Transformers** (Hugging Face) - Transformer models
- **Sentence-Transformers** - Sentence embedding models
- **SetFit** - Few-shot learning framework
- **Datasets** (Hugging Face) - Dataset management
- **NLTK** - Natural language toolkit

### Vector Search & Similarity
- **FAISS** (Facebook AI Similarity Search) - Vector similarity search
- **HNSWlib** - Hierarchical Navigable Small World graph library
- **HDScan** - Hierarchical density-based clustering

### Data Processing
- **NumPy** - Numerical computing
- **Pandas** - Data manipulation and analysis
- **NetworkX** - Graph algorithms

### Text Processing & Matching
- **RapidFuzz** - Fuzzy string matching
- **Rank-BM25** - BM25 retrieval algorithm
- **Model2Vec** - Static embedding models

## Web & API

- **Requests** - HTTP client
- **PyYAML** - YAML configuration parsing
- **JSON** - JSON handling

## CLI & Development

- **Click** - Command line interface framework
- **Rich** - Rich terminal formatting
- **Black** - Code formatter
- **Ruff** - Linter and formatter

## Testing & Quality

- **pytest** - Testing framework with async support
- **pytest-asyncio** - Async test support
- **pytest-cov** - Coverage reporting

## LLM Integration

- **litellm** - Unified LLM API (multi-provider support)
- **openai** - OpenAI SDK
- **anthropic** - Anthropic SDK

## Configuration & Build

- **pyproject.toml** - Project configuration and dependency management
- **ruff.toml** - Linting and formatting rules
- **.github/workflows** - CI/CD pipeline
